 Página Web – Seguros Fulano

 Se puede acceder a la pagina mediante servidor gratuito

  https://segurosfulano.page.gd/

Este proyecto es una página web creada como trabajo práctico.  
El objetivo principal es presentar distintos tipos de seguros y facilitar el contacto con posibles clientes.

 Descripción del proyecto
La página está pensada para la venta de servicios de seguros.  
Cuenta con 5 páginas HTML distintas, donde se explica de forma breve cada tipo de seguro para que el usuario pueda informarse y luego contactarse.

La navegación es simple y clara, enfocada únicamente en mostrar los servicios sin sobrecargar el diseño.

 Tipos de seguros
El sitio ofrece información sobre 5 tipos de seguros diferentes.  
Cada uno tiene una explicación corta y clara, orientada a que el cliente entienda el servicio y se comunique para obtener más información.

 Tecnologías utilizadas
- HTML  
- CSS  
- SASS  
- Bootstrap  

 Diseño y estilos
- Se utilizó SASS para organizar mejor los estilos  
- Se agregaron animaciones simples en los botones al pasar el mouse (hover)  
- El diseño es sobrio y no invasivo, ya que el foco está en la venta de servicios  

 Navegación
- El sitio permite navegar entre las 5 páginas HTML  
- En la versión responsive se implementó un menú hamburguesa para dispositivos móviles  

 Responsive
La página se adapta correctamente a:
- Computadoras
- Tablets
- Celulares
